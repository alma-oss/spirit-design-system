# Spirit Design System

Spirit is an open-source design system developed by [Alma Career (formerly LMC)](https://www.almacareer.com).

It provides a comprehensive set of packages for building consistent UIs across Alma Career products.

## Packages

| Package                          | Description                                                   |
| -------------------------------- | ------------------------------------------------------------- |
| `@alma-oss/spirit-analytics`     | Analytic tools for Spirit Design System                       |
| `@alma-oss/spirit-codemods`      | Codemod transforms for Spirit Design System version migration |
| `@alma-oss/spirit-design-tokens` | Design tokens for Spirit Design System                        |
| `@alma-oss/spirit-icons`         | Icons for Spirit Design System                                |
| `@alma-oss/spirit-web`           | CSS and vanilla JS implementation of Spirit Design System     |
| `@alma-oss/spirit-web-react`     | React implementation of Spirit Design System components       |
