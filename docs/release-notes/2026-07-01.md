# 2026-07-01 Spirit Design System Release Notes

📢 🚀 🎉 Ahoy, we have a new release with features and improvements. Here we go.

## General Changes

This release rounds out the file-upload experience we've been building. You can now put uploads into an interaction-only disabled state — the drop zone stays visible but users can't add or remove files — and pair that with a new upload-disabled visual state. We've also added a shake animation helper for repeated validation errors, so users notice when the same mistake happens again.

We fixed long filenames truncating incorrectly in `UNSTABLE_File` and a `Toast` stacking issue when you override z-index via CSS variables.

## What's Next 🔮

We're closing out stabilization work and pushing into v5 prep and Combobox:

- **File and FileUpload stabilization**
  `UNSTABLE_File` and `UNSTABLE_FileUpload` are on track to become stable components, with the legacy `FileUploader` removed and migration support planned.

- **Combobox (React)**
  Multi-select combobox with inline filtering for job positions, locations, and similar search-style filters.

- **Card — Employer gallery variant**
  A dedicated `Card` variant for the Jobs.cz employer gallery layout.

- **DrawerPanel composition**
  `DrawerPanelHeader` and `DrawerPanelContent` subcomponents for a clearer, composable drawer API.
