# 2026-06-02 Spirit Design System Release Notes

📢 🚀 🎉 Ahoy, we have a new release with features and improvements. Here we go.

## General Changes

This is a focused release for teams using `UNSTABLE_Picker`. You can now give the visible label and the dropdown popover different `theme` values — handy when the field sits on one surface (for example a branded header) but the menu should stay on the default light theme.

## What's Next 🔮

We're continuing the sprint focus on **FileUpload** polish and **Combobox** for search-style multi-select:

- **UNSTABLE_Combobox**
  Multi-select combobox with inline filtering (web implementation in progress; React follow-up) for job positions, locations, and similar search filters.

- **Form fields — outline variant**
  Box fields (`TextField`, `TextArea`, `Select`, `FileUploader`, `UNSTABLE_Picker`) will get a border-only `outline` variant alongside the default filled style.

- **UNSTABLE_ComboboxOption**
  React building blocks for combobox list items with correct WAI-ARIA grid structure, query highlighting, and optional dismiss actions.

- **File Upload — validation feedback**
  Optional visual hint (for example a shake animation) when the same validation situation happens again, so users notice repeated errors.
