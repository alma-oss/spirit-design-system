# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-27

### Added

- Initial release of @alma-oss/vite-plugin-spirit-icons
- Vite plugin for generating Spirit Design System icons package
- Support for SVG compilation, React component generation, and TypeScript constant generation
- Configurable paths for icon sources and output directories
- Support for multiple icon sets via plugin configuration
