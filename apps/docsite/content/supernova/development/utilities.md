## Display

Controls the display mode of an element with the [display](https://developer.mozilla.org/en-US/docs/Web/CSS/display) CSS property.

| CSS value    | All breakpoints | Tablet and larger     | Desktop and larger     |
| ------------ | --------------- | --------------------- | ---------------------- |
| block        | d-block         | d-tablet-block        | d-desktop-block        |
| flex         | d-flex          | d-tablet-flex         | d-desktop-flex         |
| inline       | d-inline        | d-tablet-inline       | d-desktop-inline       |
| inline-block | d-inline-block  | d-tablet-inline-block | d-desktop-inline-block |
| inline-flex  | d-inline-flex   | d-tablet-inline-flex  | d-desktop-inline-flex  |
| none         | d-none          | d-tablet-none         | d-desktop-none         |
