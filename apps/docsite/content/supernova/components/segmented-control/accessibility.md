## **Accessibility**

The majority of accessibility issues are already guaranteed by implementation. In the case of modifications or composition in HTML, please follow the recommendations below.

- Provide a clear visual distinction for the selected segment.

- Make sure each segment can be focused via keyboard and has sufficient contrast.

- Use icons only if they are universally understood; otherwise pair with text.
