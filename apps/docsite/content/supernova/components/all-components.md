The purpose of this table is to present the actual status of the Figma design and the components.  
Designers can verify that Figma accurately reflects the actual state of the component, and developers will find out here if the components they plan to use will appear exactly as they do in the proposed design.  
Additionally, in the event of any discrepancy between the design and the code, you will quickly identify which part is correct and which will be updated.

## Status explanation

#### Components

| Status     | Description                                                                            |
| ---------- | -------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| Stable     | a component is implemented with all features and has no known issues                   |
| Stable     | Outdated                                                                               | a component is implemented with all features but is behind Figma and needs a fix or update |
| Unstable   | a component is implemented but missing some features and will be updated in the future |
| Deprecated | a component is not supported anymore                                                   |

#### HTML | React

| Status        | Description                                                                              |
| ------------- | ---------------------------------------------------------------------------------------- |
| Up to date    | a component is available and reflects the actual design                                  |
| Outdated      | a component is available but does not reflect the actual Figma design or needs an update |
| Not available | a component is currently not available                                                   |
| Not scheduled | a component is not planned to be available for this library/framework                    |
| Deprecated    | a component is deprecated for this library/framework                                     |

#### Figma

| Status        | Description                                                   |
| ------------- | ------------------------------------------------------------- |
| Up to date    | design is available and up to date                            |
| Outdated      | design is available but needs to be updated                   |
| Not available | design is currently not available or it is not planned at all |
