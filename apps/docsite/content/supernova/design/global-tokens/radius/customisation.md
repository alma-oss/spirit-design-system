### For designers

- There is no Figma style for radii, so if you need your own borders, you must change everything manually.

- For long design system sustainability is good manually set up own radius scale with correct values. This scale should use designers and developers

- There is example of radius usage in figma compared with code

![](https://studio-assets.supernova.io/design-systems/10180/87ab8e31-648a-456d-b27c-2679cd14d8c8.png)

Are you interested in how to set up the entire brand? Check out [the guide](/latest/introduction/how-to-customize-spirit-BnwsYmmD)

### For developers

- Use only Radius values which are shown in Figma (or Supernova if you use it)

- Unfortunately figma does not support tokens, rem or em values, so you must connect px values with correct tokens

- If you need any changes, contact your designer
