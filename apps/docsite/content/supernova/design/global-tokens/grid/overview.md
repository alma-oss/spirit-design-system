#### Design Usage

Grid is the essential responsive component that should be placed on every page. On the grid are placed all components and compositions. There are these variants:

| Name   | Mobile     | Tablet     | Desktop    |
| ------ | ---------- | ---------- | ---------- |
| 1col   | 1 column   | 1 column   | 1column    |
| 2col   | 2 columns  | 2 columns  | 2 columns  |
| 3col   | 3 columns  | 3 columns  | 3 columns  |
| 4col   | 4 columns  | 4 columns  | 4 columns  |
| 5col   | 5 columns  | 5 columns  | 5 columns  |
| 6col   | 6 columns  | 6 columns  | 6 columns  |
| 12col  | 12 columns | 12 columns | 12 columns |
| Narrow | 6 columns  | 10 columns | 12 columns |
