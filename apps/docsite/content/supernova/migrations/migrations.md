Welcome to the migration guide for upgrading to the next major version of our design system.

Major version upgrades, such as moving from version 2 to version 3, often include breaking changes that must be resolved. To assist with this process, we provide React Codemods—automated tools designed to handle many of the required adjustments for you. These Codemods can significantly simplify the upgrade process for projects using Web-React package.

However, it is important to note that other packages are not supported by Codemods. For these packages, manual intervention will be necessary to resolve any breaking changes.  
In this guide, you will find detailed instructions and recommendations to help you navigate the migration process efficiently. Our goal is to ensure you have the resources and support needed to upgrade to the next major version of Spirit with confidence.
